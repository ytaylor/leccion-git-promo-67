## Comandos de Git 


 1. Clonar: 
    - `git clone <url-del-repositorio>`: Clona un repositorio remoto a tu máquina local.
2. Agregar cambios:
    - `git add -A`: Agrega todos los archivos modificados al área de preparación.
3. Confirmar cambios:
    - `git commit -m "Mensaje de commit"`: Crea un nuevo commit con los cambios agregados al área de preparación.
4. Subir cambios:
    - `git push`: Sube los cambios confirmados al repositorio remoto.
5. Bajar cambios:
    - `git pull`: Descarga los cambios del repositorio remoto y los fusiona con tu rama local.
